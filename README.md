Aircon的toolbox 图片加水印与社交媒体裁剪工具
做这个工具的原因其实很简单
经常需要把照片上传到小红书、Instagram 等平台。每次发图前都得打开 Photoshop，裁成对应尺寸，自己压缩图片再加上自己的水印真的很烦
所以干脆写了一个纯前端的小工具，把最常用的功能放在一起。
无需安装软件，打开网页即可使用。

小红书（3:4）
Instagram 正方形（1:1）
Instagram 竖版（4:5）
横版封面（16:9）

**水印请使用png格式的透明底水印**
